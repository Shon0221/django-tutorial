# 開發環境

安裝 Django

      pip3 install django
      
測試Django是否安裝

      python3 -m django --version
      
建立資料夾

      mkdir mysite
      cd mysite
      
初始化專案
  
      django-admin startproject mysite
      
啟動服務

      cd mysite
      python manage.py runserver
